What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content

How Do You Select An Element From DOM?

getElementById(): Selects an element by its unique ID. Example: document.getElementById('elementID'). querySelector(): Selects the first matching element using a CSS selector. Example: document.querySelector('.className'). getElementsByClassName() or querySelectorAll(): Selects multiple elements by class name or selector. Example: document.getElementsByClassName('className') or document.querySelectorAll('.className').

What is event Delegation in the context of DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for its child elements. It leverages event bubbling, where events propagate up the DOM tree, allowing the parent to handle events from dynamically added children. This is efficient and reduces the need for multiple listeners.

How do you Manipulate and elements Attribute and Styles Using DOM?

Manipulate Attributes: Use setAttribute(), getAttribute(), or removeAttribute() to modify an element's attributes. Example: element.setAttribute('src', 'image.jpg'). Manipulate Styles: Modify the style property directly or use classList to add, remove, or toggle CSS classes. Example: element.style.color = 'red' or element.classList.add('active').
